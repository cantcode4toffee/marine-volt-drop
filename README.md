# ⚡ Marine Volt Drop Calculator
Standards-verified volt drop calculator for marine electrical systems. Part of the Vita electrical design toolset.

### Standards basis
- ABYC E-11:2021 (TA 2023) §11.15.1.2.7 — 3% critical, 10% non-critical
- ISO 13297:2020 §5.5, §5.6, Annex A — 10% general, 3% safety-critical
- ISO 16315:DIS 2023 §10.6 — defers to ISO 13297
- ABYC E-30:2021 §30.5.1 — defers to ABYC E-11
